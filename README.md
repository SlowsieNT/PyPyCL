## Guide
Make sure `PyPy` directory contains `pypy3.exe`<br>
Make sure `PyPy.bat` and `PyPyVars.bat` are outside `PyPy` directory<br>
Looks like this:
```
PyPy
PyPy.bat
PyPyVars.bat
```
